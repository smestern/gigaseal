about this app

gigaseal is a batch electrophysiology feature-extraction toolkit for .abf and .nwb recordings. This page is a browser-based equivalent of the desktop GUI: pick a file, choose an analysis module, edit parameters, run, view results, export CSV / Excel.

two deployment profiles, one codebase

The same Flask app ships in two profiles, switched by a single env var:

  • public — caps uploads at 5 files × 50 MB, anonymous, rate-limited. The instance you're using right now.
  • lab — no caps, optional bearer-token auth, optional NFS-mounted server-side file picker. Run it on a lab VM.

self-host with Docker

git clone https://github.com/smestern/pyAPisolation
cd pyAPisolation
docker build -t gigaseal-web .
docker run -p 8000:8000 \
  -e GIGASEAL_WEB_PROFILE=lab \
  -e GIGASEAL_API_TOKEN=changeme \
  -v /path/to/abfs:/data:ro \
  -e GIGASEAL_SERVER_PATH_ROOT=/data \
  gigaseal-web

architecture

The analysis logic itself is unchanged: every job runs through gigaseal.analysis.run_batch, the same entry point the desktop GUI uses. The web layer is a thin shell — session storage, a job queue with a concurrency cap, and a Plotly / Tabulator frontend served by Flask.
